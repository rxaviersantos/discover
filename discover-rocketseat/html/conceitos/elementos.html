# HTML - Linguagem de Marcação de Hipertexto

Pricipais elementos especiais da marcação HTML

###head

O elemento HTML providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.

###title

O elemento HTML título define o título do documento, mostrado na barra de título de um navegador ou na aba da página.

###body

Representa o conteúdo de um documento HTML

###header

Representa um grupo de suporte introdutório ou navegacional, pode conter alguns elementos de cabeçalho mas também outros elementos como logo, seções de cabeçalho, formulário de pesquisa e outros.

###footer

Reprersenta um rodapé, contendo informaçõessobre o autor da seção de dados, direitos autorais ou links para documentos relacionados

###article

Representa uma composição independente em um documento, página, aplicação, ou site, ou que é destinado a ser distribuido de forma independente ou reutilizável, por exemplo, em sindicação.

###section

Representa uma seção genérica contida em um documento HTML, geralmente com um título, quando não existir um elemento semântico mais específico para representá-lo.

###p

Representa um parágrafo

###div

É um container genérico para conteúdo de fluxo, que de certa forma não representa nada. Ele pode ser utilizado para agrupar elementos para fins de estilos (usando class ou id), ou porque eles compartilham valores de atributos, como lang.

###span

É um conteiner generico em linha para conteúdo fraseado , que não representa nada por natureza. Ele pode ser usado para agrupar elementos para fins de estilo (usando os atributos class ou id ), ou para compartilhar valores de atributos como lang.

###img

Representa a inserção de imagem no documento, sendo implementado também pelo HTML5 para uma melhor experiência com o elemento
